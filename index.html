OdinLibrary
A state-driven JavaScript application that manages a collection of books using constructor functions and prototype methods. Books are dynamically created via a modal form and rendered from a centralized data array to ensure clean separation between data and UI. Each book instance maintains a unique identifier for reliable state manipulation, including removal and read-status toggling. The project emphasizes DOM–data synchronization, event handling, and scalable front-end architecture.
